# **App Name**: Token Time Machine

## Core Features:

- Token Input: Input field for the token name/symbol.
- Date Selection: Date picker to select the purchase date.
- Purchase Price Input: Input field to enter the purchase price of the token.
- Profit/Loss Calculation: Use an AI tool to fetch current price of the token and calculate the potential profit/loss.
- Result Display: Display the calculated profit/loss in a clear format.

## Style Guidelines:

- Primary color: Dark blue (#1A237E) for a professional look.
- Secondary color: Light grey (#E0E0E0) for contrast.
- Accent: Green (#4CAF50) to indicate profit and red (#F44336) to indicate loss.
- Clean and readable sans-serif fonts.
- Simple and modern icons for token and date selection.
- Clean and responsive layout.
- Subtle animations for loading states and result transitions.
