## 🎯 What This Is

Token Time Machine is the **most addictive crypto calculator** on the internet. Users input:
- Any cryptocurrency name or symbol
- A date in the past 
- An investment amount

And discover exactly how much their investment would be worth today. The results are often **life-changing** (and sometimes heart-breaking).

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/yourusername/token-time-machine

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your API keys (Firebase, DexGuru, etc.)

# Start the development server
npm run dev
```

Visit `http://localhost:9002` to see the magic happen! 🎪
